FOLLOW THE FOLLOWING STEPS TO RUN THE CODE:

1. Open the terminal where all the lex and yacc files are present.
2. Enter the following command
    "flex cucu.l"  (without the quotes and press enter)
 
    "bison -d cucu.y"    (without the quotes and press enter)
    
    "cc lex.yy.c cucu.tab.c -o cucu"  (without the quotes and press enter)


3. Now to run your given cu program, enter
     "./cucu filename.cu"  (without the quotes and press enter)
